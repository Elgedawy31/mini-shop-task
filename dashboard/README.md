# Dashboard

Vite + React + TypeScript dashboard for the mini shop workspace.

## Scripts

```bash
bun dev
bun build
bun preview
bun test
```

Runs on http://localhost:5000 and talks to the API at http://localhost:5001.

## Structure

- `src/features` - feature-owned pages, hooks, services, and types.
- `src/shared` - reusable UI, hooks, services, and utilities.
- `src/router` - route composition.

Project-level linting, formatting, and Husky live in the repository root.
