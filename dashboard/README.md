# Dashboard

Vite + React + TypeScript dashboard for the mini shop workspace.

## Scripts

```bash
bun dev
bun build
bun preview
bun test
```

The dashboard runs on http://localhost:5000.

## Structure

- `src/features` - feature-owned pages, hooks, services, and types.
- `src/shared` - reusable UI, hooks, services, and utilities.
- `src/router` - route composition.

Project-level linting, formatting, Husky, and Docker Compose live in the repository root.
